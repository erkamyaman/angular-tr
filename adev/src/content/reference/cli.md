# CLI Referansı

| Komut                              | Takma Ad   | Açıklama                                                                                                   |
| :--------------------------------- | :--------- | :--------------------------------------------------------------------------------------------------------- |
| [`add`](cli/add)                   |            | Projenize harici bir kütüphane desteği ekler.                                                              |
| [`analytics`](cli/analytics)       |            | Angular CLI kullanım metriklerinin toplanmasını yapılandırır.                                              |
| [`build`](cli/build)               | `b`        | Bir Angular uygulamasını veya kütüphanesini, verilen çıktı yolundaki dist/ adlı bir çıktı dizinine derler. |
| [`cache`](cli/cache)               |            | Kalıcı disk önbelleğini yapılandırır ve önbellek istatistiklerini alır.                                    |
| [`completion`](cli/completion)     |            | Terminaliniz için Angular CLI otomatik tamamlamayı ayarlar.                                                |
| [`config`](cli/config)             |            | Çalışma alanı için angular.json dosyasındaki Angular yapılandırma değerlerini alır veya ayarlar.           |
| [`deploy`](cli/deploy)             |            | Belirtilen proje veya çalışma alanındaki varsayılan proje için dağıtım oluşturucusunu çağırır.             |
| [`e2e`](cli/e2e)                   | `e`        | Bir Angular uygulamasını derler ve sunar, ardından uçtan uca testleri çalıştırır.                          |
| [`extract-i18n`](cli/extract-i18n) |            | Kaynak kodundan i18n mesajlarını çıkarır.                                                                  |
| [`generate`](cli/generate)         | `g`        | Bir şematik temelinde dosyalar oluşturur ve/veya değiştirir.                                               |
| [`lint`](cli/lint)                 |            | Belirli bir proje klasöründeki Angular uygulama kodu üzerinde lint araçlarını çalıştırır.                  |
| [`new`](cli/new)                   | `n`        | Yeni bir Angular çalışma alanı oluşturur.                                                                  |
| [`run`](cli/run)                   |            | Projenizde tanımlanmış isteğe bağlı özel oluşturucu yapılandırmasıyla bir Architect hedefini çalıştırır.   |
| [`serve`](cli/serve)               | `s`, `dev` | Uygulamanızı derler ve sunar, dosya değişikliklerinde yeniden derler.                                      |
| [`test`](cli/test)                 | `t`        | Bir projede birim testlerini çalıştırır.                                                                   |
| [`update`](cli/update)             |            | Çalışma alanınızı ve bağımlılıklarını günceller. [Güncelleme Kılavuzu](/update-guide)'na bakın.            |
| [`version`](cli/version)           | `v`        | Angular CLI sürümünü çıktılar.                                                                             |
