# Functions should be invoked in text interpolation.

Bu tanılama, metin interpolasyonlarında çağrılmamış fonksiyonları tespit eder.

```angular-ts
import {Component} from '@angular/core';

@Component({
  template: `{{ getValue }}`,
})
class MyComponent {
  getValue() {
    return 'value';
  }
}
```

## What's wrong with that?

Metin interpolasyonlarındaki fonksiyonlar bir değer döndürmek için çağrılmalıdır.
Fonksiyon çağrılmazsa, herhangi bir değer döndürmez ve interpolasyon beklendiği gibi çalışmaz.

## What should I do instead?

Değeri döndürmek için metin interpolasyonunda kullandığınızda fonksiyonu çağırdığınızdan emin olun.

```angular-ts
import {Component} from '@angular/core';

@Component({
  template: `{{ getValue() }}`,
})
class MyComponent {
  getValue() {
    return 'value';
  }
}
```

## Configuration requirements

Herhangi bir genişletilmiş tanılamanın verilmesi için [`strictTemplates`](/tools/cli/template-typecheck#strict-mode) etkinleştirilmelidir.
`uninvokedFunctionInTextInterpolation`, `strictTemplates` dışında ek bir gereksinime sahip değildir.

## What if I can't avoid this?

Bu tanılama, projenin `tsconfig.json` dosyası düzenlenerek devre dışı bırakılabilir:

```json
{
  "angularCompilerOptions": {
    "extendedDiagnostics": {
      "checks": {
        "uninvokedFunctionInTextInterpolation": "suppress"
      }
    }
  }
}
```

Daha fazla bilgi için [genişletilmiş tanılama yapılandırması](/extended-diagnostics#configuration) bölümüne bakın.
