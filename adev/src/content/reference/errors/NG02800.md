# HttpClient yapılandırmasında JSONP desteği

Angular, `HttpClient` yapılandırmasında gerekli destek sağlanmadan bir `JSONP` isteği yapmaya çalıştığınızda bu hatayı üretir.
`JSONP` desteğini etkinleştirmek için aşağıdakilerden birini yapabilirsiniz:

- `bootstrapApplication` kullanıldığında `provideHttpClient` fonksiyon çağrısı sırasında `withJsonpSupport()` fonksiyonunu argüman olarak ekleyin (örn. `provideHttpClient(withJsonpSupport())`)
- NgModule tabanlı başlatma kullanıldığında kök AppModule'ünüzde `HttpClientJsonpModule`'ü import edin.

## Hatayı ayıklama

Yukarıda açıklandığı gibi, JSONP desteğinin uygulamanıza `withJsonpSupport` fonksiyonu çağrılarak (`provideHttpClient` kullanıldığında) veya `HttpClientJsonpModule` modülü import edilerek eklendiğinden emin olun.
