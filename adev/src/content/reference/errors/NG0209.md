# Geçersiz multi sağlayıcı

Angular çalışma zamanı, `multi: true` ile kullanılması amaçlanan bir token enjekte ettiğinde ancak bunun yerine bir Array olmayan değer bulduğunda bu hatayı fırlatır. Örneğin, `ENVIRONMENT_INITIALIZER` şu şekilde sağlanmalıdır: `{provide: ENVIRONMENT_INITIALIZER, multi: true, useValue: () => {...}}`.
