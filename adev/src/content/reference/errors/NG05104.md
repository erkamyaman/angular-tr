# Root element was not found

Başlatılan bileşenler, bir `@NgModule` dekoratörünün `bootstrap` özelliğinde veya bağımsız bileşenler için `bootstrapApplication` fonksiyonunun ilk parametresi olarak tanımlanır.

Bu hata, Angular bu bileşenlerden birini başlatmaya çalıştığında ancak DOM'da karşılık gelen düğümü bulamadığında oluşur.

## Debugging the error

Bu sorun, seçici etiketle eşleşmediğinde oluşur

```typescript
@Component({
  selector: 'my-app',
  ...
})
export class App {}
```

<!-- prettier-ignore -->
```html
<html>
  <app-root></app-root> <!-- Doesn't match the selector -->
</html>
```

Etiketi doğru olanla değiştirin:

<!-- prettier-ignore -->
```html
<html>
  <my-app></my-app> <!-- OK -->
</html>
```
