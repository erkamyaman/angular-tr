# Missing Reference Target

<docs-video src="https://www.youtube.com/embed/fUSAg4kp2WQ"/>

Angular, `{{ PLACEHOLDER }}` dışa aktarım adıyla bir direktif bulamıyor.
Bu, eksik bir import veya bir direktifteki eksik bir [`exportAs`](api/core/Directive#exportAs) ile yaygındır.

HELPFUL: Bu, yaygın bir çalışma zamanı hatası olan [NG0301: Export Not Found](errors/NG0301)'un derleyici karşılığıdır.

## Debugging the error

Bulunamayan dışa aktarımı kullanan şablonları veya modülleri izlemek için dışa aktarım adının dize adını kullanın.

Tüm bağımlılıkların Modüllerimize doğru şekilde import edildiğinden ve tanımlandığından emin olun.
Örneğin, bulunamayan dışa aktarım `ngForm` ise, eksik dışa aktarım hatasını çözmek için `FormsModule`'ü import etmemiz ve `*.module.ts` dosyasındaki imports listesinde tanımlamamız gerekir.

```ts

import { FormsModule } from '@angular/forms';

@NgModule({
…
imports: [
FormsModule,
…

```

Yakın zamanda bir import eklediyseniz, bu değişiklikleri görmek için sunucunuzu yeniden başlatmanız gerekecektir.
