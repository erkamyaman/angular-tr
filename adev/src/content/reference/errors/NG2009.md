# Geçersiz Shadow DOM seçicisi

`ViewEncapsulation.ShadowDom` kullanan bir bileşenin seçicisi, özel element etiket adı gereksinimlerini karşılamıyor.

Bir etiket adının geçerli bir özel element adı olarak kabul edilmesi için şunları sağlaması gerekir:

- Küçük harf olmalı.
- Bir tire içermeli.
- Bir harf \(a-z\) ile başlamalı.

## Hatayı ayıklama

Bileşeninizin seçicisini gereksinimleri karşılayacak şekilde yeniden adlandırın.

**Önce:**

```ts
@Component({
  selector: 'comp',
  encapsulation: ViewEncapsulation.ShadowDom
…
})

```

**Sonra:**

```ts
@Component({
  selector: 'app-comp',
  encapsulation: ViewEncapsulation.ShadowDom
…
})
```
