# Yanlış Asenkron Doğrulayıcı Dönüş Türü

Asenkron doğrulayıcılar bir promise veya observable döndürmeli ve doğrulama başarısız olsa da başarılı olsa da bunları emit etmeli/resolve etmelidir. Özellikle [AsyncValidatorFn API](api/forms/AsyncValidator)'sini uygulamaları gerekir.

```typescript
export function isTenAsync(control: AbstractControl): Observable<ValidationErrors | null> {
  const v: number = control.value;
  if (v !== 10) {
    // Doğrulama hatası içeren bir nesne yay.
    return of({'notTen': true, 'requiredValue': 10});
  }
  // Hata oluşmadığını belirtmek için null yay.
  return of(null);
}
```
