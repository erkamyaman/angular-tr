# Argüman Sabit Değil

Angular derleyicisinde metadata çıkarımını daha hızlı hale getirmek için `@NgModule`, `@Pipe`, `@Component`, `@Directive` ve `@Injectable` dekoratörleri argüman olarak yalnızca nesne sabitleri (object literal) kabul eder.

Bu, View Engine'e kıyasla dekoratörler için daha katı argüman gereksinimleri uygulayan [Ivy'deki kasıtlı bir değişikliktir](https://github.com/angular/angular/issues/30840#issuecomment-498869540).
Ivy bu yaklaşımı gerektirir çünkü dekoratörleri, ifadeleri sınıf çıktısındaki diğer konumlara taşıyarak derler.

## Hatayı ayıklama

Tüm tanımlamaları:

```ts
const moduleDefinition = {…}

@NgModule(moduleDefinition)
export class AppModule {
  constructor() {}
}

```

dekoratörün içine taşıyın:

```ts

@NgModule({…})
export class AppModule {
  constructor() {}
}

```
