# Invalid Element

Element HTML spesifikasyonu tarafından tanımlanmadığından veya bu element seçicisine sahip bir bileşen ya da direktif olmadığından, bir veya daha fazla element derleme sırasında çözümlenemiyor.

HELPFUL: Bu, yaygın bir çalışma zamanı hatası olan `NG0304: '${tagName}' is not a known element: ...`'in derleyici karşılığıdır.

## Debugging the error

Elementin kullanıldığı dosyayı/dosyaları bulmak için hatadaki element adını kullanın.

Adın ve seçicinin doğru olduğunu kontrol edin.

Bileşenin NgModule'ünüz veya bağımsız bileşeniniz içinde doğru şekilde import edildiğinden, `imports` alanındaki varlığını kontrol ederek emin olun. Bileşen bir NgModule'de tanımlanmışsa (yani bağımsız değilse), `exports` alanındaki varlığını kontrol ederek doğru şekilde dışa aktarıldığından emin olun.

Özel elementler veya web bileşenleri kullanırken, uygulama modülüne [`CUSTOM_ELEMENTS_SCHEMA`](api/core/CUSTOM_ELEMENTS_SCHEMA) eklediğinizden emin olun.

Bu sorunu çözmezse, kullandığınız dışa aktarımlar ve özellikler için import edilmiş kütüphanelerde son yapılan değişiklikleri kontrol edin ve sunucunuzu yeniden başlatın.
