# Export Not Found

<docs-video src="https://www.youtube.com/embed/fUSAg4kp2WQ"/>

Angular, `{{ PLACEHOLDER }}` dışa aktarım adıyla bir direktif bulamıyor. Dışa aktarım adı, direktif dekoratörünün `exportAs` özelliğinde belirtilir. Bu, şablonlarda FormsModule veya Material modüllerini kullanırken ve karşılık gelen modülleri import etmeyi unuttuğunuzda yaygındır.

HELPFUL: Bu, yaygın bir derleyici hatası olan [NG8003: No directive found with export](errors/NG8003)'ün çalışma zamanı karşılığıdır.

## Debugging the error

Bu dışa aktarımı kullanan şablonları veya modülleri izlemek için dışa aktarım adını kullanın.

Tüm bağımlılıkların doğru şekilde import edildiğinden ve NgModule'lerinizde tanımlandığından emin olun. Örneğin, bulunamayan dışa aktarım `ngForm` ise, hatayı çözmek için `FormsModule`'ü import etmemiz ve `*.module.ts` dosyasındaki imports listesinde tanımlamamız gerekir.

```ts

import { FormsModule } from '@angular/forms';

@NgModule({
…
imports: [
FormsModule,
…

```

Yakın zamanda bir import eklediyseniz, bu değişiklikleri görmek için sunucunuzu yeniden başlatmanız gerekebilir.
